Statistics API Reference

This page documents all statistics available in Relevent.jl.

Dyad-Level Statistics

Statistics based on the detailed interaction history between the focal sender-receiver pair.

Relevent.PriorInteraction — Type
PriorInteraction <: AbstractStatistic

Statistic for prior interaction between sender and receiver. Uses half-life decay.

Fields

  • halflife::Float64: Half-life for decay
  • direction::Symbol: :outgoing, :incoming, or :both

Relevent.LocalInertia — Type
LocalInertia <: AbstractStatistic
